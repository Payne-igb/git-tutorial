git cheatsheet

1. git checkout {commit id} - This is will take you back to the previous commit of git with the git id, and will erase all changes made after that time.
2. git add . - This adds the root directory of the file to git to be watched.
3. git add {file-name} - This adds the file to git to be watched.
4. git commit -m {message} - This commits changes to git with the particular message explaining the commit.
5. git log - Shows all the commits made previously and the commit id, with the user who committed the changes.
6. git init - To initialize a git repository.
7. ls - To list all the files under the current directory.
8. git status - To check if there's any change to be committed into git.
9. git log --all - Shows all the commmit history with the ones made ahead that were previously cleared by using `git checkout {commit id}`
10. git log --all --graph - Shows all the commmit history in form of branches.
